# Logging::Facade

I kept rewriting a simple, logging provider/facade so I just decided to make it its own, little project.

## Installation

Add this line to your application's Gemfile:

    gem 'logging-facade'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install logging-facade
